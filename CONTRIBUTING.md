# Contributing to PromptTap

Thank you for your interest in contributing to PromptTap!

## Issues

- Bug reports
- Feature requests
- UX improvements
- Documentation fixes

are all welcome.

Before creating a new issue, please check whether a similar issue already exists.

---

## Pull Requests

1. Fork the repository
2. Create a feature branch
3. Commit your changes
4. Open a Pull Request

Please keep PRs focused and reasonably small.

---

## Code Style

- Follow standard Swift and SwiftUI conventions
- Prefer simple and readable implementations
- Keep the native macOS UX in mind

---

## License and Relicensing

By submitting a contribution (including code, documentation, assets, or other materials),
you agree that your contribution may be relicensed, redistributed, or incorporated into
future versions of PromptTap under different license terms by the project maintainer.

This helps ensure the long-term sustainability and future flexibility of the project.
